Core implementation of the descriptor.

This function should compute the descriptor only for environments in descriptor.environments and computing only features in descriptor.features. By default, these would correspond to the environments and features coming from Descriptor::environments and Descriptor::features respectively; but this can be overrode to only compute them on a subset though Descriptor::compute_partial.
